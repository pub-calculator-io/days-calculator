The free days calculator makes it easy to figure out how many days it has been since some date. This day tracker can include or exclude weekends and holidays.

[https://www.calculator.io/days-calculator/](https://www.calculator.io/days-calculator/)

== Usage ==

Add the Days Calculator shortcode to your page, post or sidebar:

`[ci_days_calculator]`

Add the following code to your template where you would like the Days Calculator to appear:

`<?php display_ci_days_calculator(); ?>`

== Installation ==

1. Upload the Days Calculator /ci_days_calculator/ folder to the /wp-content/plugins/ directory.
2. Activate the Days Calculator plugin through the "Plugins" menu in WordPress.
